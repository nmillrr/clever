# Changelog

All notable changes to the Academic Access Extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-06-13

### Added
- Initial release of Academic Access Extension
- Paywall detection for major academic publishers
- Institution proxy configuration
- Automatic DOI and article metadata extraction
- Access button for institutional proxy redirection
- Unpaywall API integration for open access alternatives
- Access history tracking
- Multi-institution support
- Radix UI implementation for consistent design system
- Dark/light mode support
- Privacy-focused design with local storage only

### Technical
- Manifest V3 compliant extension
- Modular component architecture
- Webpack bundling system
- Comprehensive documentation
